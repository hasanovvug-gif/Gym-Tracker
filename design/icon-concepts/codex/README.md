# Codex — концепты иконки

Все варианты построены на формах из мобильного интерфейса Gym Tracker и используют только две заливки: `#0B0C0E` и `#C8F031`.

| Вариант | Суть | Риск |
|---|---|---|
| 1 · G Lock | Нарисованная с нуля узкая «G» с плоскими срезами — типографический характер Oswald без шрифта. | Без контекста буква не сразу сообщает, что приложение про тренировки. |
| 2 · Rest Gate | Толстое незамкнутое кольцо повторяет главный визуальный момент rest-таймера. | Может читаться как общий индикатор активности или загрузки. |
| 3 · Set Stack | Три плашки показывают состояния подхода: готово, в процессе, впереди. | Самый продуктовый, но и самый абстрактный знак. |

Основные файлы — лайм на почти-чёрном; `-inverse` — почти-чёрное на сплошном лайме. Все SVG имеют `viewBox="0 0 1024 1024"`, не используют текст, шрифты, фильтры или градиенты.
