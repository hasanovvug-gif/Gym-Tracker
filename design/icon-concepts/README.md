# Концепты иконки приложения

Четыре варианта, показаны Вугару 2026-07-22. **Выбор не сделан** — продолжение в новой сессии.

SVG в этой папке — тёмная заливка (лайм `#C8F031` на `#0B0C0E`). У каждого есть
инверсная версия: те же формы чёрным (`#0B0C0E`) на сплошном лайме — при сборке
финального ассета выбирается одна из двух заливок.

| Файл | Вариант | Суть | Риск |
|------|---------|------|------|
| `a-barbell.svg` | A · Штанга | Читается мгновенно | Стоковая метка, таких в сторе десятки |
| `b-monogram.svg` | B · Монограмма «G» | Тот же Oswald, что в заголовках экранов — иконка и приложение как одна вещь | Буква без предмета менее «спортивна» |
| `c-rest-ring.svg` | C · Кольцо отдыха | Взято прямо с экрана тренировки, своё | «Спортзал» читается не сразу |
| `d-sets.svg` | D · Подходы | Три полосы: сделано / в процессе / впереди. Живёт в мелком размере | Абстрактно, без подсказки не прочитать |

Рекомендация Клода — **B**: единственный вариант, продолжающий типографику самого
приложения, а не накладывающий поверх чужой значок.

## Что делать после выбора

1. Отрисовать выбранный в 1024×1024 PNG → `mobile/assets/images/icon.png`
2. Adaptive-иконка под Android: форма отдельным слоем + фон заливкой
   (`mobile/assets/images/adaptive-icon.png`)
3. Splash: та же форма по центру, фон `#0B0C0E` (тёмная) / `#F4F5F3` (светлая) —
   `expo-splash-screen` уже настроен в `mobile/app.json`
4. Проверить в симуляторе и на домашнем экране в мелком размере

Шрифт Oswald для варианта B: `mobile/node_modules/@expo-google-fonts/oswald/700Bold/`.
В SVG буква задана текстом — перед экспортом перевести в кривые, иначе на машине
без Oswald подставится другой шрифт.
