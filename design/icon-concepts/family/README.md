# Gym Tracker · family icon concepts

Все четыре варианта продолжают грамматику AsbestosGuard: сплошная тёмная плитка, белая контурная геометрия и один лаймовый линейный знак. Белый и лаймовый штрихи везде имеют одинаковую толщину `96` при стороне `1024`.

| Вариант | Что заявляет | В чём риск | Родство с AsbestosGuard |
|---|---|---|---|
| V1 · Diamond sets | Три состояния подходов: короткий маркер, основной рабочий подход, следующий маркер | На 60 px верхний и нижний штрихи становятся почти точками; это самый плотный вариант | прямое |
| V2 · Diamond bar | Тренировочный гриф, сведённый к одной линии с симметричными воротниками | Может читаться как абстрактный технический узел, а не как гриф без контекста | прямое |
| V3 · Rest ring | Незамкнутый прогресс rest-таймера; лаймовая капсула сохраняет подпись семейства | Круглая форма дальше от прямого ромбового родства | по грамматике |
| V4 · Hex frame | Силовая, устойчивая рамка с прямыми гранями; новый член той же геометрической серии | Шестиугольник менее специфичен для фитнеса, чем гриф или таймер | по грамматике |

## Общая система

- Фон: `#0B0C0E`
- Контур: `#FFFFFF`
- Акцент: `#C8F031`
- Толщина обоих штрихов: `96`
- Скругление плитки: `213`
- Внешняя геометрия и штрихи остаются внутри центральных 80% с учётом толщины линии
- В SVG нет текста, градиентов, теней и внешних ресурсов
